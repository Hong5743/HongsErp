## Agent skills

### Issue tracker

Issues live in GitHub Issues (gh CLI). External PRs are not a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Default mattpocock/skills label vocabulary (needs-triage, needs-info, ready-for-agent, ready-for-human, wontfix). See `docs/agents/triage-labels.md`.

### Domain docs

Multi-context repo: CONTEXT-MAP.md at root points to per-context CONTEXT.md files. See `docs/agents/domain.md`.
